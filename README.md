# Contact Manager

A simple CLI application to manage contacts with add, view, search, and delete functionality.

## Features
- Add new contacts (name, phone, email)
- View all contacts
- Search contacts by name
- Delete contacts
- Persist data to file

## Usage
```bash
python contact_manager.py
```

## Tech Stack
- Python 3.10+
- File I/O for data persistence

## What I Learned
- Working with file operations
- Dictionary and list manipulation
- User input handling
- Basic CRUD operations
